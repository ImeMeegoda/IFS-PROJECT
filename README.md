# Kngu Flower Shop

A simple landing page for a floral shop built with HTML, CSS, and Bootstrap.

## Files
- `index.html` — homepage layout
- `styles.css` — custom page styles
- `images/` — image assets used on the page

## How to use
1. Open `index.html` in a browser.
2. Or serve the folder with a local web server.

## Notes
- Uses Bootstrap 5 for layout and responsive grid.
- Uses Google Fonts and Font Awesome for typography and icons.
